**BlueSynth**
A Synth VST Project with AI sound-design features by David Thomas, Ellis Day, Adithya Prashanth and Mihir Borkar, all current students at the University of California - Irvine
Working with SoundEthics (soundethics.org) and AI@UCI

**Overview:**
BlueSynth is a 64-voice polyphonic synthesizer plugin with oscillator waveform selection and ADSR envelope control. The main goal is to have an LLM model that can assist users to sound design the sounds that they want & need.

**How It's Made:**
Tools used: C++, JUCE, Xcode
