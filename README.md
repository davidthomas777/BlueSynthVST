**BlueSynth: VST**

A Synth VST Project with AI sound-design features created by David Thomas, Ellis Day, Adithya Prashanth, and Mihir Borkar, all current undergraduate students at the University of California - Irvine

Working with SoundEthics (soundethics.org) and AI@UCI

**Overview:**

BlueSynth is a 64-voice polyphonic synthesizer plugin with oscillator waveform selection and ADSR envelope control. The main goal is to have an LLM model that can assist users by curating the sounds they want & need.

**How It's Made:**

Tools used: C++, C, JUCE, Xcode
